Overview
A web service to analyze Kakaotalk chat data and show user with various edition of it.
카카오톡 대화를 분석하여 웹페이지에 게시하는 웹 서비스.
Tech stack
Python, Flask
  • For the light project, Flask was better than Django, which I had used.
  • Had a personal purpose to learn about Flask.
  • 가벼운 크기의 프로젝트에는 Flask가 적절했다.
  • 개인적인 Flask 공부 목적도 있음.
Contributions
  • Pre-processed over 1 million lines of Kakaotalk chat data and create a SQL table using Pandas.
  • Made the analyzed data visible in web with Flask.
  • Developing a time capsule letter service.
  • Pandas 이용해서 백만 줄 이상의 카카오톡 데이터를 전처리하고, SQL 테이블 생성.
  • Flask를 통해 분석된 데이터를 웹에 게시.
