Overview
A pixel art editor that can draw pixel arts and share them.
픽셀아트를 그리고 공유, 저장할 수 있는 모바일 에디터
Tech stack
C#, Unity
  • 팀 합류 이전 Android 환경으로 개발하면서 픽셀 아트 그리기 기능에 Unity가 더 효과적임을 깨달음.
  • 합류 이후 주 기능을 원활히 하기 위해 Unity로 개발하고, Unity에서 지원하는 C# 스크립트를 이용함.
Firebase
  • 게임 기능이 있는 프로젝트의 특성상 비동기 작업을 수행하기 위해 적용.
  • Unity와의 호환성 역시 높다.
Git
  • For collaboration.
  • 협업을 위해 적용한 툴.
Contributions
  • Developed key features such as user authentication, community forums, and pixel art sharing using C# in the Unity environment, enhancing user engagement.
  • Integrated Firebase’s asynchronous communication capabilities for real-time updates, enabling seamless data synchronization between database and the game application.
  • Initially planned to implement the community on the web environment, but faced a double-login issue with session/cookies. To resolve this, integrated user management and community features within Unity, adding pixel art download functionality to enhance service usability.
  • 사용자, 커뮤니티에서 필요한 API 구현 및 데이터베이스 구조 설계 & 관리
  • Firebase의 비동기 통신 기능을 이용하여 게임과 데이터베이스 간 실시간 데이터 동기화를 구현
  • 웹페이지로 커뮤니티를 구현하려 했으나 세션/쿠키로도 두 번의 로그인이 필요. 이를 해결하기 위해 Unity 내에서 사용자 관리와 커뮤니티 기능을 구현하고, 픽셀아트 다운로드 기능을 추가하여 서비스 활용도를 높임.
